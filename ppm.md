# PPM `Portable Pixel Map`
- When we have a program to generate a rectangular picture specifing color of each pixel.

## PP3
```ppm
P3
# comment line
100 200
255
```
- pp3 specifies this is PPM ASCII file.
- 3rd, width height
- 4th, mostly 255, max intenstiy of picture

## PP6
```ppm
P6
# comment line
100 200
255
```
